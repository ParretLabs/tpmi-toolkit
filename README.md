# tpmi-toolkit
Tools &amp; Algorithms for AI-Generated TagPro Maps

# Run
```bash
# Clone the repository
git clone https://github.com/ParretLabs/tpmi-toolkit.git
cd tpmi-toolkit

# Install the packages
npm i
# or
yarn

# Then run one of the examples in the /examples directory
cd /examples/basic
node index.js
```
